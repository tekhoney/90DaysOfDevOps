Day 03 – Linux Commands Practice

Today’s goal is to build your Linux command confidence.

You will create a cheat sheet of commands focused on:

Process management

ps - this command is use to check all the running process  syntax: ps - <option>

top - this is use to check the resource consupmption by the processes, Syntax: top

Kill - To kill the running process. Syntax: kill -9 <PID> (-9 is used to terminate the process forcefully)

File system
cd - change directory

cd .. - to get back to the previous directory

mkdir <dir1> : to create the directory 

rmdir: to delete the directory

su - : to switch user

vim : to edit the file 

cat to read the file content and also can create the file.

touch: to update the time stamp 

mv: to move or rename the file or directory name. 

cp: to copy the file or directory 

head -10 : it will show the first 10 lines of the file 

tail -10 : it will show the last 10 lines of the file. 

grep : it is to find in the file system. 


Networking troubleshooting

ping ip address or host name: to check the connectivity 

ipaddr : to check the ip address 


This is the command toolkit you will reuse for years.
